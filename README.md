---

http://echarts.baidu.com

## Install

```
$ spm install jui-echarts --save
```

## Usage

```js
var echarts = require('jui-echarts');
var myChart = echarts.init(document.getElementById('main'));
var option = {};
myChart.setOption(option);

```
